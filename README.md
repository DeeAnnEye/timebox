# ⏱️ Timebox

A minimal Electron-based calendar & reminder app focused on simplicity.

Timebox provides a clean monthly calendar view and lets you add simple text reminders to any date. All data is stored locally.

## Features

- Monthly calendar view
- Highlights today automatically
- Add reminders to dates
- Visual indicator for dates with reminders
- Offline-first (local storage)

## Tech Stack

- Electron
- React + TypeScript
- CSS Grid
- localStorage

## Run Locally

```bash
npm install
npm run dev
